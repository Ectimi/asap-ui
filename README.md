# asap-ui

本项目是基于 Varlet 组件库的 React 版本，
为了方便开发者快速开发，
我们提供了一套基于 React 的组件库，
目前组件库还在不断完善中，
欢迎大家 pr。

组件样式，Api 与 Varlet 保持一致
组件库地址：https://varlet.gitee.io/varlet-ui/#/zh-CN/home

# 贡献流程

## 提供 cli 工具

```bash
$ yarn global add asap-ui-cli   # 全局安装cli
```

```bash
  $ asap-ui-cli create <component-name>  # 创建组件
```
